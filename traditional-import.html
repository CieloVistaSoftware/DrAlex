Chat Widget Traditional Import Test

This page demonstrates how the chat widget is included using a traditional script tag:

<script src="js/chat-widget.js"></script>

The chat-widget.js file automatically initializes itself when the DOM is loaded.

How it Works

  1. The script is loaded with a regular script tag
  2. It contains a ChatWidget class
  3. At the bottom of the file, it adds a DOM content loaded event listener
  4. When the page loads, it automatically creates and initializes the widget
