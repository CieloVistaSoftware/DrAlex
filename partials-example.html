HTML Partials Example

Reusable Components for Dr. Alex Kisitu's Website

This page demonstrates how to use HTML partials with JavaScript

How It Works

1. HTML Structure

Create placeholder elements in your HTML for partials

<div id="header-placeholder"></div>
<div id="navigation-placeholder"></div>
<div id="footer-placeholder"></div>

2. JavaScript Loading

Load partials with fetch() when the page loads

loadPartial('partials/header.html', 'header-placeholder');
loadPartial('partials/navigation.html', 'navigation-placeholder');
loadPartial('partials/footer.html', 'footer-placeholder');

3. Modular Components

Each partial serves a specific purpose:

  • header.html - Page head and site container
  • navigation.html - Main site navigation menu
  • footer.html - Site footer with copyright and social links

Benefits of This Approach

  • No server-side languages required (no PHP)
  • Works with static website hosting
  • Easy to maintain and update
  • Consistent look and feel across all pages
  • Reduced duplication of code
  • Separation of concerns - navigation is now its own component
