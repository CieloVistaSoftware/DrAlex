Website Structure

The website is organized with a multi-page structure, following this layout:

DrAlex/ ├── css/ │ └── styles.css # All styles for the entire website ├── js/ │ └── scripts.js # All JavaScript functionality ├── images/ # Images folder │ ├── DrAlex.jpeg │ ├── CyberPunk.png │ └── [other images] ├── index.html # Home page ├── about.html # About page ├── services.html # Services page ├── portfolio.html # Portfolio/Patient Stories page ├── contact.html # Contact page └── readme.html # This documentation file
Note: This multi-page structure makes it much easier to maintain compared to a single large file. Each page focuses on one specific section of content.

Basic HTML Editing

To edit text content, locate the text you want to change in the HTML file and modify it directly. For example:

<!-- Before -->
<p class="site-subtitle">Personal Eye Care just for you!</p>

<!-- After -->
<p class="site-subtitle">Comprehensive Eye Care for the Whole Family!</p>
Tip: Use a code editor like Visual Studio Code, Notepad++, or Sublime Text for easier HTML editing.

Modifying the Style (CSS)

All styling is centralized in css/styles.css. This makes it easy to maintain a consistent look across all pages.

Common Style Changes

Changing Colors

The website uses CSS variables for colors, making it easy to update the entire color scheme by changing just a few values at the top of the CSS file:

:root {
  /* Theme colors with purple secondary */
  --primary: #6366f1;     /* Change this to update primary color */
  --secondary: #9333ea;   /* Change this to update secondary color */
  --accent: #10b981;      /* Change this to update accent color */
  /* ... other color variables ... */
}

Adjusting Fonts

To change the font family, locate this section in the CSS:

body {
  font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif;
  /* ... other body styles ... */
}

Replace with your desired font (you may need to add a link to Google Fonts in the HTML head section).

Working with JavaScript

All JavaScript functionality is contained in js/scripts.js. The main functions are:

  • Theme Management: Controls dark/light mode and color themes
  • Navigation: Highlights active page in navigation
  • Back to Top: Manages the back-to-top button functionality
Warning: Be careful when editing JavaScript. Always make a backup before making changes, and test thoroughly after any modifications.

Adding New Pages

To add a new page to the website:

  1. Create a new HTML file in the root directory (e.g., new-page.html)
  2. Copy the basic structure from an existing page as a template
  3. Update the page title, meta description, and keywords
  4. Replace the content with your new content
  5. Add a link to the new page in the navigation section of all existing pages:
<nav id="site-nav" class="site-nav">
  <ul id="nav-list" class="nav-list">
    <li><a href="index.html" class="nav-link">Home</a></li>
    <li><a href="about.html" class="nav-link">About</a></li>
    <li><a href="services.html" class="nav-link">Services</a></li>
    <li><a href="portfolio.html" class="nav-link">Portfolio</a></li>
    <li><a href="contact.html" class="nav-link">Contact</a></li>
    <li><a href="new-page.html" class="nav-link">New Page</a></li> <!-- Add this line -->
  </ul>
</nav>

Managing Images

All images should be stored in the images/ folder. To add a new image:

  1. Optimize the image for web (reduce file size while maintaining quality)
  2. Place the image file in the images/ folder
  3. Reference it in your HTML using the relative path: 
    <img src="images/your-image-name.jpg" alt="Descriptive alt text">
Best Practice: Always include meaningful alt text for images to improve accessibility and SEO.

SEO Best Practices

For optimal search engine performance:

  1. Page Titles: Each page has a unique, descriptive title tag that includes "Dr. Alex Kisitu"
  2. Meta Descriptions: Each page has a unique meta description (150-160 characters)
  3. Keywords: Relevant keywords are included in meta keywords tag and naturally throughout content
  4. Headings: Proper use of heading hierarchy (H1 → H2 → H3)
  5. Image Alt Text: All images have descriptive alt text

When updating content, remember to keep these SEO elements in mind.

FAQ and Troubleshooting

The styling isn't updating after I changed the CSS

Try clearing your browser cache (Ctrl+F5 or Cmd+Shift+R) or using a private/incognito window to view the site.

Images aren't displaying

Check that the image path is correct and that the image exists in the images folder. Image paths are case-sensitive.

Navigation links aren't working

Verify that the href attributes in your navigation links point to the correct filenames.

I want to change the theme colors site-wide

Edit the CSS variables at the top of the css/styles.css file.

How do I add a new section to an existing page?

Copy the structure of an existing section and modify the content as needed:

<section id="new-section" class="content-section">
  <h2 class="section-title">New Section Title</h2>
  <div class="section-content">
    <p>Your content here...</p>
  </div>
</section>
